# Advanced C Coding

This repository contains advanced C programming assignments for the FPC Advanced Classroom 1st Year course.

- **set01/** to **set05/**: Assignment sets containing C programming exercises and problems
- **problem01_debug_log.txt** and **problem02_debug_log.txt**: Debug logs for specific problems
- **.gitignore**: Git ignore file
